## 角色分工

| 阶段 | 分工 |
|------|------|
| 需求访谈 | PM 与 boss 结构化访谈，输出访谈纪要 |
| 需求分析 + PRD确认 | PM 与 boss 直接沟通，PRD 由 PM 确认 |
| 接口设计 | **BE 负责输出 API 文档**，FE 确认接口文档后冻结 |
| UI 设计 | UI Agent 执行，**boss 确认审美**后才能推进 |
| FE 开发 | **UI设计稿 + API接口文档 双确认后**才能调度 FE 开始 |
| 具体实现（代码/测试） | PM 调度对应 agent 执行 |
| 运营物料 | PM 调度 OP agent 执行，**boss 审核** |
