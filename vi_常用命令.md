# Vi/Vim编辑器常用命令整理

## 一、基本概念

### 1. 三种模式
- **命令模式（Command Mode）**：默认模式，用于导航、编辑和执行命令
- **插入模式（Insert Mode）**：用于输入文本，按 `i`、`a`、`o` 等键进入
- **末行模式（Last Line Mode）**：用于执行保存、退出等命令，按 `:` 进入

## 二、命令模式常用命令

### 1. 光标移动
- `h` 或 左箭头：左移一格
- `j` 或 下箭头：下移一行
- `k` 或 上箭头：上移一行
- `l` 或 右箭头：右移一格
- `0`（数字零）：移动到行首
- `$`：移动到行尾
- `^`：移动到当前行的第一个非空白字符
- `w`：向前移动一个单词
- `b`：向后移动一个单词
- `e`：移动到单词的末尾
- `G`：移动到文件末尾
- `gg`：移动到文件开头
- `nG`：移动到第n行（如 `5G` 移动到第5行）

### 2. 文本操作
- `x`：删除当前字符
- `nx`：删除从当前位置开始的n个字符
- `dd`：删除当前行
- `ndd`：删除从当前行开始的n行
- `D`：删除从当前位置到行尾的所有字符
- `yy`：复制当前行
- `nyy`：复制从当前行开始的n行
- `p`：粘贴到当前光标之后
- `P`：粘贴到当前光标之前
- `u`：撤销上一步操作
- `Ctrl + r`： redo 上一步操作
- `r`：替换当前字符
- `R`：进入替换模式，连续替换字符

### 3. 搜索和替换
- `/pattern`：向前搜索 `pattern`
- `?pattern`：向后搜索 `pattern`
- `n`：继续搜索下一个匹配项
- `N`：继续搜索上一个匹配项
- `:%s/old/new/g`：全局替换（末行模式命令）

## 三、插入模式常用命令

### 1. 进入插入模式
- `i`：在当前光标位置前插入
- `I`：在当前行的行首插入
- `a`：在当前光标位置后插入
- `A`：在当前行的行尾插入
- `o`：在当前行下方新开一行插入
- `O`：在当前行上方新开一行插入

### 2. 退出插入模式
- `Esc`：退出插入模式，返回命令模式

## 四、末行模式常用命令

### 1. 文件操作
- `:w`：保存文件
- `:w filename`：另存为 filename
- `:q`：退出编辑器（未修改时）
- `:q!`：强制退出，不保存修改
- `:wq` 或 `ZZ`：保存并退出
- `:x`：保存并退出（如果文件有修改）

### 2. 窗口操作
- `:split` 或 `:sp`：水平分割窗口
- `:vsplit` 或 `:vsp`：垂直分割窗口
- `Ctrl + w + w`：在窗口间切换
- `Ctrl + w + h/j/k/l`：在窗口间切换
- `Ctrl + w + q`：关闭当前窗口

### 3. 其他命令
- `:set number` 或 `:set nu`：显示行号
- `:set nonumber` 或 `:set nonu`：隐藏行号
- `:set syntax on`：开启语法高亮
- `:set list`：显示不可见字符（如换行符、制表符）

## 五、高级技巧

### 1. 多文件编辑
- `:e filename`：在当前窗口打开新文件
- `:bn`：切换到下一个文件
- `:bp`：切换到上一个文件
- `:ls`：列出所有打开的文件

### 2. 宏录制
- `q{寄存器}`：开始录制宏到指定寄存器
- `q`：停止录制
- `@{寄存器}`：执行录制好的宏
- `n@{寄存器}`：执行n次宏

### 3. 文本块操作
- `v`：进入字符可视化模式
- `V`：进入行可视化模式
- `Ctrl + v`：进入块可视化模式
- `y`：复制选中的文本块
- `d`：删除选中的文本块
- `c`：删除选中的文本块并进入插入模式

## 六、Vim配置

### 1. 临时配置
在末行模式下执行的配置只对当前会话有效

### 2. 永久配置
在用户主目录下创建 `.vimrc` 文件（Windows系统为 `_vimrc`），加入配置选项

示例配置：
```vim
set number        "显示行号
set tabstop=4     "设置制表符宽度为4个字符
set shiftwidth=4  "设置自动缩进宽度为4个字符
set expandtab     "将制表符转换为空格
set syntax on     "开启语法高亮
set ruler         "显示光标位置
set wrap          "自动换行
set linebreak     "按单词换行
set showcmd       "显示命令
set laststatus=2  "显示状态栏
```

## 七、常用快捷键总结

| 功能 | 命令 |
|------|------|
| 保存 | :w |
| 退出 | :q |
| 保存并退出 | :wq 或 ZZ |
| 强制退出 | :q! |
| 显示行号 | :set nu |
| 隐藏行号 | :set nonu |
| 查找 | /pattern |
| 替换 | :%s/old/new/g |
| 撤销 | u |
| 重做 | Ctrl + r |
| 复制 | yy |
| 粘贴 | p |
| 删除行 | dd |

## 八、帮助命令

- `:help`：打开Vim帮助系统
- `:help [topic]`：查看特定主题的帮助（如 `:help dd`）
- `:h`：`help` 命令的简写
