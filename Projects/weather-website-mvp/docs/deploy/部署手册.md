# 部署手册

## 环境要求

- **操作系统**：Ubuntu 20.04+ / CentOS 7+
- **Node.js**：v18+
- **Nginx**：latest
- **域名/端口**：后端占用 3000，Nginx 监听 80/443

---

## 部署结构

```
[用户浏览器]
    ↓
[Nginx:80/443]  →  静态文件（前端页面）
    ↓ 反向代理
[Node.js:3000]  →  后端服务（天气 API）
```

---

## 步骤一：上传代码

```bash
# 在服务器上创建项目目录
mkdir -p /var/www/weather-website
cd /var/www/weather-website

# 上传后端代码（code/backend/*）到 server/ 子目录
mkdir -p server

# 上传前端代码（code/frontend/*）到 public/ 子目录
mkdir -p public
```

---

## 步骤二：安装依赖（后端）

```bash
cd /var/www/weather-website/server
npm install
```

---

## 步骤三：配置环境变量（可选）

创建 `.env` 文件（可选，Mock 模式无需填写）：

```bash
# .env
PORT=3000
NODE_ENV=production
# WEATHER_API_KEY=your_api_key_here   # 未来接真实API时填写
```

---

## 步骤四：启动后端服务

```bash
# 直接启动（开发环境）
node code/backend/server.js

# 使用 PM2 守护进程（生产环境）
npm install -g pm2
pm2 start code/backend/server.js --name weather-api

# 查看状态
pm2 status

# 开机自启
pm2 save
pm2 startup
```

验证后端是否正常：
```bash
curl http://localhost:3000/api/weather?city=北京
```

---

## 步骤五：配置 Nginx

编辑 `/etc/nginx/sites-available/weather-website`：

```nginx
server {
    listen 80;
    server_name your-domain.com;

    root /var/www/weather-website/public;
    index index.html;

    # 前端静态文件
    location / {
        try_files $uri $uri/ /index.html;
    }

    # 反向代理到后端 API
    location /api/ {
        proxy_pass http://127.0.0.1:3000/api/;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

启用站点：
```bash
sudo ln -s /etc/nginx/sites-available/weather-website /etc/nginx/sites-enabled/
sudo nginx -t          # 检查配置
sudo systemctl reload nginx
```

---

## 步骤六：配置 HTTPS（可选）

```bash
sudo certbot --nginx -d your-domain.com
```

---

## 常用命令汇总

| 操作 | 命令 |
|------|------|
| 启动后端 | `node server.js` |
| PM2 启动 | `pm2 start server.js --name weather-api` |
| 重启后端 | `pm2 restart weather-api` |
| 查看日志 | `pm2 logs weather-api` |
| 停止后端 | `pm2 stop weather-api` |
| Nginx 重载配置 | `sudo systemctl reload nginx` |
| Nginx 重启 | `sudo systemctl restart nginx` |
| 查看端口占用 | `lsof -i :3000` |

---

## 环境变量说明

| 变量名 | 必填 | 说明 | 默认值 |
|--------|------|------|--------|
| `PORT` | 否 | 后端服务端口 | `3000` |
| `NODE_ENV` | 否 | 运行环境 | `development` |
| `WEATHER_API_KEY` | 否 | 真实天气 API Key（暂未使用） | 空 |
