# 产品介绍

## 一句话介绍

一款森系可爱风格的天气预报小程序，告诉你今天怎么穿、出门注意什么。

---

## 核心卖点

1. **三日报完整呈现** — 昨天、今天、明天三天气温/天气一目了然，不用来回切换日期
2. **智能穿搭推荐** — 根据温度和天气状况，秒给搭配建议，不用动脑子想穿什么
3. **温馨提示提醒** — 紫外线/雨伞/空气质量，有雷点提前告知，出门前心里有底
4. **森系颜值设计** — 森林绿 + 奶油白配色，大圆角温暖视觉，体验舒适不疲劳
5. **覆盖 12 座城市** — 北上广深 + 成都/杭州/重庆/武汉/西安/南京/苏州/天津

---

## 目标用户

- **上班族**：出门前快速决策穿什么、看是否带伞
- **学生党**：早八/上课前 30 秒搞定穿搭参考
- **轻度需求用户**：不想装厚重天气 App，只想一个页面搞定出门准备
