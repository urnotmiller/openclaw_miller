# 功能说明

---

## 功能清单

| 功能 | 说明 |
|------|------|
| **城市切换** | 支持北京/上海/广州/深圳/杭州/成都/重庆/武汉/西安/南京/苏州/天津，共12城 |
| **三日预报** | 展示昨天、今天、明天三天天气，含温度区间、天气类型、图标 |
| **穿搭推荐** | 根据当日温度区间推荐上衣/裤子/鞋子/配件，支持「换一批」重新生成 |
| **温馨提示** | 展示紫外线指数、是否需要带伞、空气质量等提醒信息 |
| **换穿搭按钮** | 一键刷新穿搭推荐列表，每次给出不同搭配方案 |

---

## 使用场景

### 场景一：早起上班族
> 7:30 起床，看一眼天气 → 发现今天降温3度 → 穿搭推荐给出「厚外套+长裤」→ 顺手点「换一批」换一个风格 → 出门

### 场景二：临时外出
> 下午收到邀约出门 → 切换到目标城市 → 看明日预报决定带不带外套 → 温馨提示提醒晚上有雨 → 顺手带上伞

### 场景三：旅行出发前
> 去南京出差 → 切换城市查看未来三天天气 → 发现后天降温 → 提前把厚衣服装进箱子

## 数据说明

- 天气数据来源：后端 Node.js 服务（Mock 数据，端口 3000）
- 前端为纯静态页面，可直接由 Nginx 托管
- 前后端通过 RESTful JSON 接口通信
