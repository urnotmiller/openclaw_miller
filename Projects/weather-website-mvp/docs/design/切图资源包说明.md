## 3. 技术要求

### 3.1 图标规范
- **格式**：SVG 优先（支持多分辨率），PNG 作为兜底（提供 @2x 版本）
- **命名**：全部小写，单词间用连字符 `-`
- **颜色**：SVG 内使用 `currentColor`，通过 CSS 控制颜色
- **可访问性**：所有图标 `<svg>` 标签需带 `aria-hidden="true"` 并提供可见文本标签

### 3.2 图片规范
- **格式**：WebP（优先），JPEG 作为兜底
- **图片压缩**：JPEG 质量 80%，PNG 使用 TinyPNG 压缩
- **命名**：全部小写，单词间用连字符 `-`

### 3.3 字体
- **中文**：系统字体栈（PingFang SC / Hiragino Sans GB / Microsoft YaHei / Noto Sans SC）
- **英文/数字**：Nunito（Google Fonts），备选 SF Pro Display
- **字重**：400（Regular）、500（Medium）、600（Semibold）、700（Bold）

## 5. 实现建议

### 5.1 图标库选型（前端开发参考）
- **推荐**：[Phosphor Icons](https://phosphoricons.com/)（风格圆润，支持多种粗细）
- **备选**：Remix Icon、Lucide Icons
- **天气图标**：可使用 [Weather Icons](https://erikflowers.github.io/weather-icons/) 或自行 SVG

### 5.2 CSS 动画实现（纯 CSS，无需图片）
- 太阳旋转：`@keyframes spin { to { transform: rotate(360deg); } }`
- 云朵飘动：`@keyframes float { 0%,100% { transform: translateX(0); } 50% { transform: translateX(8px); } }`
- 雨滴下落：`@keyframes rain { 0% { transform: translateY(-100%); opacity: 0; } 10% { opacity: 1; } 100% { transform: translateY(100vh); opacity: 0; } }`
- 雪花飘落：同上改为随机横向偏移
- 叶子旋转加载：`@keyframes leafSpin { to { transform: rotate(360deg); } }`
