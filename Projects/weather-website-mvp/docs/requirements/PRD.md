## 一句话介绍

一款森系可爱风格的天气预报小程序，告诉你今天怎么穿、出门注意什么。

---

## 核心功能需求

### 1. 城市天气查询
- 支持切换城市
- 支持城市：北京、上海、广州、深圳、杭州、成都、重庆、武汉、西安、南京、苏州、天津（共12城）
- 默认展示用户定位城市（或默认北京）

### 2. 三日报展示
- 同时展示**昨天、今天、明天**三天天气
- 每日报含：温度区间、天气类型、天气图标

### 3. 穿着搭配推荐
- 根据当日温度区间，自动推荐穿着搭配
- 包含：上衣、裤子、鞋子、配件
- 支持「换一批」重新生成不同搭配方案

### 4. 温馨提示
- 展示紫外线指数
- 提醒是否需要带伞
- 空气质量提示
- 其他天气注意事项

## 非功能性需求

### 设计风格
- **主题**：森系可爱风
- **品牌色**：森林绿 `#8DB48E` + 奶油白 `#FFF8F0`
- **圆角**：16-24px 大圆角
- **字体**：PingFang SC / Nunito
- **动画**：fadeInUp / scaleUp / 天气动画

### 技术要求
- 前端：纯 HTML + CSS + JavaScript（无框架依赖）
- 后端：Node.js / Express
- 数据：RESTful JSON 接口
- 响应式：移动优先
